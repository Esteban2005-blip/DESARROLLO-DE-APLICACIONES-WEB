# Galería Interactiva con JavaScript

Este proyecto consiste en una galería de imágenes interactiva desarrollada con HTML, CSS y JavaScript, que permite agregar, seleccionar y eliminar imágenes dinámicamente mediante la manipulación del DOM.

## Funcionalidades
- Agregar imágenes mediante URL
- Seleccionar una imagen con clic
- Resaltar imagen seleccionada
- Eliminar imagen seleccionada con botón o tecla Delete

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
