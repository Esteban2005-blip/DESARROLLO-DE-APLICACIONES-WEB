# Sitio Web Responsivo con Bootstrap 5

Este proyecto consiste en una página web moderna y adaptable creada con Bootstrap 5 y CSS3.

## Tecnologías Utilizadas
- HTML5
- CSS3
- Bootstrap 5
- GitHub Pages

## Características
- Diseño responsivo
- Uso de Grid y Flexbox
- Componentes interactivos
- Estilos personalizados
- Animaciones y transiciones

## Publicación
El sitio fue publicado usando GitHub Pages desde la rama main.
